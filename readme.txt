 What can 123Aurora do for you ? ───────────────────────────────────────────────

  123Aurora is a set of PHP/JS files that will create a web solar logger for Power One Aurora Inverters.
    
 Prerequisites ────────────────────────────────────────────────────────────────
 
  123Aurora rely on the communication application from Curtronics, which is not included in this package.
 
 Installation ─────────────────────────────────────────────────────────────────
 
 - Install the communication application for Aurora inverters from Curtronics. (http://www.curtronics.com/Solar/AuroraData.htm)
 - Extract 123Aurora on your web server's folder. 
 - Start the administration interface (scripts/123aurora admin)
 - Launch 123Aurora as root (scripts/123aurora.sh start)
